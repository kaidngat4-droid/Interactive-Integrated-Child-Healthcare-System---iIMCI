# نظام الرعاية التكاملية التفاعلي لصحة الطفل - iIMCI
# Interactive Integrated Child Healthcare System - iIMCI

![Yemen Eagle Logo](assets/images/yemen-eagle.png)

نظام رقمي تفاعلي ذكي لإدارة برنامج الرعاية المتكاملة لصحة الطفل (iIMCI) في المرافق الصحية.

## 🎯 الهدف
تحسين تسجيل وتصنيف ومتابعة حالات الأطفال بطريقة رقمية وسريعة، وتقليل الاعتماد على الملفات الورقية.

## 🏥 المميزات
- تسجيل بيانات الطفل بالكامل (الاسم، العمر، الوزن، الطول، الحرارة، علامات الخطورة)
- التصنيف التلقائي والمعالجة الذكية وكذلك وسيلة للتدريب حسب إرشادات IMCI وبروتوكولات منظمة الصحة العالمية واليونيسيف وتجنب التشخيص الخاطئ وكذلك المعالجات الخاطئة
- توفير الجهد للعاملين الصحيين والأطباء عن طريق الترحيل التلقائي للبيانات إلى التقرير الشهري:
  - السعال
  - الإسهال
  - الحمى
  - علامات الخطورة العامة
  - علاج الحميات والملاريا
  - سوء التغذية
  - التطعيمات ومواعيد إعطائها
  - مشاكل الأذن والحلق
- لوحة تحكم للمرافق الصحية لعرض الأطفال المسجلين
- تقارير شهرية تلخص الحالات والإحالات
- حفظ البيانات محلياً (LocalStorage) مع خطة الربط بقاعدة بيانات مركزية مستقبلاً
- نظام تقييم التغذية والتواصل للطفل (< سنتين)
- إشعارات بالعلاج والعلاجات الموصى بها

## 🛠️ خطة التطوير القادمة
- ربط النظام بقاعدة بيانات Firebase أو سيرفر محلي لتخزين البيانات بشكل آمن
- إضافة نظام تسجيل دخول وصلاحيات للمستخدمين (Admin, Doctor, Nurse)
- إنشاء تقارير متقدمة قابلة للطباعة أو التصدير
- دعم التشغيل دون اتصال بالإنترنت مع مزامنة لاحقة
- تحسين واجهة المستخدم للهواتف والأجهزة اللوحية
